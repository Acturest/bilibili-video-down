# bilibili-video-down
这是一个下载bilibili视频的程序,是本人学习python模块过程中的一个小练习

ui界面使用的是python内置的tkinter

支持使用带有av号的链接,支持单独下载音频
# 使用方式
  1.下载最新版的软件包
  
  2.运行ui.exe

  注:不登录最高720P,登录可下载1080P,暂时不支持大会员即更高的清晰度
